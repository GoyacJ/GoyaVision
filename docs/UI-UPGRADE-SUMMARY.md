### 阴影系统

```css
--shadow-sm: 0 2px 8px rgba(0, 0, 0, 0.05);          /* 输入框 */
--shadow-md: 0 4px 12px rgba(102, 126, 234, 0.15);   /* 悬停 */
--shadow-lg: 0 8px 32px rgba(31, 38, 135, 0.15);     /* 卡片 */
--shadow-xl: 0 20px 60px rgba(31, 38, 135, 0.2);     /* 对话框 */
--shadow-primary: 0 8px 24px rgba(102, 126, 234, 0.4); /* 彩色阴影 */
```

### 圆角系统

```css
--radius-sm: 8px;    /* 小按钮、标签 */
--radius-md: 12px;   /* 输入框、普通按钮 */
--radius-lg: 16px;   /* 卡片、对话框 */
--radius-xl: 20px;   /* 登录框、大卡片 */
```

### 动画系统

```css
/* 淡入动画 */

/* 右滑入动画 */

/* 脉冲动画 */
@keyframes pulse { /* 2s infinite */ }

/* 浮动动画 */
@keyframes float { /* 20s infinite ease-in-out */ }
```

## 🔧 技术实现

### 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - 类型安全
- **Element Plus** - 组件库
- **Vite** - 构建工具
- **Pinia** - 状态管理

### 核心技术

#### 1. CSS 变量系统

**优势**:
- 统一主题管理
- 快速全局修改
- 易于维护扩展
- 支持动态切换（深色模式）

#### 2. Glassmorphism（磨砂玻璃效果）

```css
background: rgba(255, 255, 255, 0.95);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.3);
```

**兼容性**:
- Chrome 76+
- Safari 9+
- Firefox 103+
- Edge 79+

#### 3. 渐进增强

```css
@supports not (backdrop-filter: blur(20px)) {
  .glassmorphism {
    background: rgba(255, 255, 255, 1); /* 回退方案 */
  }
}
```
